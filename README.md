# 🤖 AI React App (MERN)

A full-stack AI-powered web application built using the MERN stack.
Users can ask AI-based questions, save responses..

---

##  Live Links

*  Frontend: https://ai-react-app-v93z.vercel.app/
*  Backend: https://ai-react-app.onrender.com

---

##  Tech Stack

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: MongoDB
* API:  AI MODEL USE OPENROUTER/AUTO

##  How to Run Locally

### 1. Clone Repository

```
git clone https://github.com/abdulrazaq1835/AI-react-app.git
```

---

### 2. Backend Setup

```
cd backend
npm install
npm start
```

---

### 3. Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

##  Environment Variables

### Frontend (.env)

```
VITE_API_URL=http://localhost:5000
```

### Backend (.env)

```
PORT=5000
MONGO_URI=your_mongodb_uri
```
